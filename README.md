# UOLab-Management-System
联合开放实验室管理系统，Java Swing + My
## 🚀 功能特性
- 用户管理：用户账号的增删改查、启用/禁用
- 成员管理：实验室成员信息管理、多条件查询、统计
- 公开课管理：课程发布、查询
- 参赛管理：竞赛记录录入、获奖统计
- 创新学分管理：学生申请、教师审核、学分统计

## 🛠️ 技术栈
- 开发语言：Java 8
- 界面框架：Swing
- 数据库：MySQL 8.0
- 版本控制：Git
